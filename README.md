### 上手指南

用搜索引擎搜索Gihub Pages 搭配Hexo 搭建个人博客


###### 开发前的配置要求

1. 一台个人电脑，有浏览器功能。
2. 能够独立安装、部署各种当前设备所需环境。如：python、Hexo、git......等

###### **安装步骤**

1. 创建自己的仓库并命名。
2. clone到自己的电脑上。
3. 安装部署搜索到的教程上需要的环境(一般只需要git和hexo)。
4. 本地配置调试个人博客页面配置文件。
5. 同步到自己github仓库里(一般教程里都有方法)。
